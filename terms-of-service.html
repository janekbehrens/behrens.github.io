2. Description of the App

Visual Progress Tracker is a Jira Cloud app that adds customizable progress bar fields to Jira issues. The App provides:

  • Three custom field types: Manual, Status-based, and Subtask-based progress bars.
  • A dashboard gadget for project-wide progress overview.
  • An admin configuration page for status mappings and color thresholds.
