Three field types

Track progress the way your team actually works

Manual

Manual Progress Bar

Drag a slider to set any percentage from 0 to 100%. Simple, flexible, and always under your control.

  • Design, content, and creative work
  • Tasks with subjective milestones
  • Quick updates during standups
Status-Based

Status Progress Bar

Automatically maps your Jira workflow statuses to percentages. Zero extra clicks — moves when the issue moves.

  • Teams with well-defined workflows
  • Sprint tracking at a glance
  • Consistent, automated reporting
Subtask

Subtask Progress Bar

Automatically calculates (resolved subtasks ÷ total subtasks). Works out of the box — no setup needed.

  • Epics and stories with subtasks
  • QA checklists and multi-step tasks
  • Onboarding flows with child issues

Also included

  • Color-coded bars — red → yellow → green as progress increases
  • Configurable color thresholds in Admin Settings
  • Progress columns in Jira list view and backlog
  • Dashboard gadget — project-wide progress overview table
  • Subtask details expandable view on each issue
  • JQL-compatible field values for search and filtering

Project-wide visibility in list view

Add progress bars as columns to any Jira list view or backlog. Scan your entire project in seconds — no opening individual issues.

  • All three field types visible as columns
  • Color-coded bars stay consistent across views
  • Spot red bars immediately — find at-risk work fast
  • Dashboard gadget for team leads and stakeholders

Project health at a glance

The Progress Tracker Overview dashboard gadget gives team leads and stakeholders a live summary table of every issue in a project — with progress values front and center.

  • Add to any Jira dashboard in seconds
  • Three calculation modes: Manual, Status, or Subtask
  • Sortable table — spot lagging items instantly
  • Data refreshes on demand or auto-caches for speed

Setup

Running in 3 steps

1

Install

Get the app from the Atlassian Marketplace and confirm the installation on your Jira site.

2

Add Fields

Create one or more progress bar custom fields in Jira Settings and add them to your project screens.

3

Done

Progress bars appear on your issues immediately. Status and Subtask types calculate automatically — no further setup.
