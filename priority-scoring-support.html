Priority Scoring — Support

Send a support request

For questions, bug reports, or feature requests, send an e-mail to the address below. Please include your Jira site URL, a brief description of the issue, and — if applicable — whether it relates to the RICE, WSJF, or ICE fields, the dashboard, or the Rovo AI agent.

support@janekbehrens.de

Support hours & response time

Support is available Monday – Sunday, including holidays (MEZ/CEST).
Target response time: within 24 hours of your message.

For critical issues (app completely unavailable or incorrect scores written to issues), please write URGENT in the subject line — these are prioritised.

Common topics

When contacting support, the following information helps us resolve your issue faster:

Custom fields: Which framework (RICE / WSJF / ICE) is affected, and whether the field is missing from the issue view or showing incorrect values.

Rovo AI agent: Which action was triggered (suggest / apply / bulk), the configured apply mode, and the exact response or error message shown in Rovo chat.

Dashboard / gadget: Whether the project has been opened in the dashboard at least once (required to build the health cache).
