Privacy Policy

Last updated: February 15, 2026

1. Introduction

This Privacy Policy explains how Visual Progress Tracker ("the App"), developed by Janek Behrens ("we", "us", "our"), handles data when installed on your Atlassian Jira Cloud instance. We are committed to protecting your privacy and being transparent about our data practices.

2. Data We Access

The App accesses the following Jira data solely to provide its functionality:

  • Issue data: Issue keys, summaries, statuses, subtask relationships, and resolution states — used to calculate progress percentages.
  • Project data: Project keys and names — used to scope progress calculations and the dashboard gadget.
  • Custom field values: Progress bar field values created by the App — used to display and update progress bars.

3. Data We Store

The App stores the following configuration data using Atlassian Forge Storage (hosted and managed by Atlassian within your cloud instance):

  • Status-to-progress mapping (e.g., "In Progress" = 50%)
  • Color threshold settings (e.g., red below 34%, green above 67%)
  • App preferences (review prompt state, install date)

We do not store any issue content, personal data, or user information.

4. Data We Do NOT Collect

  • We do not collect, transmit, or store any personally identifiable information (PII).
  • We do not send any data to external servers, third-party services, or analytics platforms.
  • We do not use cookies, tracking pixels, or any form of user tracking.
  • We do not access user emails, names, or account details beyond what Jira provides in API responses for issue data.

5. Data Processing Location

All data processing occurs entirely within the Atlassian Forge runtime environment. The App runs on Atlassian's infrastructure as part of the Runs on Atlassian program. No data leaves the Atlassian platform.

6. Third-Party Services

The App does not integrate with or send data to any third-party services. There are no external API calls, no analytics services, and no advertising networks.

7. Data Retention

  • Configuration data (status mappings, thresholds) is retained in Forge Storage as long as the App is installed. Uninstalling the App removes all stored data.
  • Custom field values are stored by Jira as standard custom field data. Removing the custom fields from Jira deletes these values.

8. GDPR Compliance

Since we do not collect or process any personal data, GDPR data subject requests (access, rectification, erasure, portability) are not directly applicable to the App. However, uninstalling the App will remove all App-related configuration data.

If you have any GDPR-related concerns, please contact us at the email address below.

9. Security

The App follows Atlassian Forge security best practices:

  • Runs in a sandboxed environment with no egress to external networks.
  • Uses scoped Jira API permissions (only reads/writes issue and field data).
  • Does not store credentials, tokens, or secrets.

10. Children's Privacy

The App is a business productivity tool and is not directed at children under 16. We do not knowingly collect data from children.

11. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last updated" date above. Continued use of the App after changes constitutes acceptance of the updated policy.
